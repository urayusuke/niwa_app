## GitHub Pages の有効化手順

ランディングページ（`docs/index.html`）を公開するには、リポジトリの設定で以下を行う。

1. GitHub リポジトリページで **Settings** タブを開く
2. 左サイドバーの **Pages** を選択
3. **Source** で「Deploy from a branch」を選択
4. **Branch** で `main` を選び、フォルダを `/docs` に変更
5. **Save** をクリック

しばらくするとページが公開され、URL が表示される。
